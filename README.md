# Mood Sensor App

Aplikasi mobile berbasis Flutter yang memanfaatkan berbagai sensor perangkat untuk menampilkan kondisi lingkungan dan aktivitas pengguna secara real-time dengan tampilan minimalis dan warna pastel.

## 📱 Fitur

* Accelerometer (mendeteksi gerakan/percepatan perangkat)
* Gyroscope (mendeteksi rotasi perangkat)
* Magnetometer (menentukan arah seperti kompas)
* GPS Location (menampilkan koordinat lokasi pengguna)
* Battery Status (menampilkan persentase baterai)
* Network Connectivity (mengetahui jenis koneksi internet)
* Device Information (informasi perangkat)
* Camera (mengambil foto langsung dari aplikasi)
* Status Otomatis (menentukan kondisi pengguna berdasarkan sensor)

## 🛠️ Teknologi yang Digunakan

* Flutter
* Dart

## 📦 Package/Depedencies yang Digunakan
* sensors_plus
* geolocator
* battery_plus
* connectivity_plus
* device_info_plus
* camera

## 📌 Tujuan

Aplikasi ini dibuat untuk memenuhi tugas implementasi sensor pada perangkat mobile menggunakan Flutter, sekaligus sebagai eksplorasi penggunaan berbagai sensor dalam satu aplikasi.
